Shell

The shell is the layout component which wraps an application. It contains a menu bar for navigation. For demonstration purposes, all examples listed here will use the relative position mode of the shell.

Base Example


Positioning Additional Content

Before Tabs

The garudaShellBarStart directive can be used to position content before the tabs.


After Tabs

The garudaShellBarEnd directive can be used to position content after the tabs.


Relative Position

By default, the position of the bar is fixed and a padding is placed on top of the content. If you don't want this behavior, you can set the [relativePosition] property to true.
