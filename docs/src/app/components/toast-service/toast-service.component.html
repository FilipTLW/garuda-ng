Message Toast

The message toast service allows you to show short notifications to the user.

The message toast service requires the app to use the Shell in order to function properly.

Base Example


Severity

The message toast has multiple severities: info, success, warn, error, contrast, secondary.


Positioning

The message toast can be positioned differently. Following values are allowed: top-left, top-center, top-right, bottom-left, bottom-center, bottom-right, center. Default value is top-right.
